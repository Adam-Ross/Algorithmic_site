Ransom Note

Enough with the MATH!! Right? Let's move on to working with some strings. In this problem we will have to figure out if a given set of letters contains the letters necessary to make a ransom note.


For example: let's say we're given an array of letters like:


['a', 'g', 'f', 'c', 't']


And what we want to find out is if the array contains the letter to make the word 'cat.' For the word 'cat' in this example our algorithm should return => true. Try it on your own before you watch the video. Good luck.
